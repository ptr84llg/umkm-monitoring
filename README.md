# monitoring-umkm

Fondasi awal proyek Laravel 13 untuk sistem monitoring UMKM berbasis data dan visual analitik interaktif.

## Catatan tahap ini
- Fokus tahap 1: struktur awal proyek, konfigurasi dasar, dan folder modular.
- Belum ada modul bisnis lengkap, migration domain lanjutan, atau API operasional final.
- Instalasi paket Laravel penuh dapat dijalankan ulang bila akses jaringan tersedia.

## Langkah lanjutan yang direkomendasikan
1. Jalankan `composer install` saat akses jaringan tersedia.
2. Salin konfigurasi environment: `cp .env.example .env`.
3. Generate app key: `php artisan key:generate`.
4. Siapkan database lalu jalankan migrasi awal sesuai kebutuhan domain.
5. Mulai server lokal dengan `php artisan serve` untuk verifikasi bootstrap aplikasi.
